    Basics of Html:

    Html stands for Hyper Text Markup Language. Basically this language is used for writing the text in web pages. This language is used in web development and gives a structure to the web page. We will discuss all the basic tags of html and use this language for making our websites.

    Importance of Html:

    Html is a basic language language for web development. Html plays very important role in making websites. There are multiple html tags which are used to perform different tasks. A website is a combination of Html,css and javascript. Html is a foundation of a website which holds the data that tells the browser what content, including text, links, and image locations, is on the page.
